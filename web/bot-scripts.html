Bot 脚本使用文档

说明：本文档提供了 bot.sh（适用于 Linux/macOS）和 bot.bat（适用于 Windows）脚本的使用指南，帮助您快速与企业微信机器人服务器进行交互。

脚本获取方式

自动获取：当您成功注册机器人后，系统会自动生成包含您的机器人 ID 和安全码的脚本文件，并发送到您的企业微信群中。

手动配置：您也可以下载通用脚本模板，手动配置您的机器人信息。

1. bot.sh (Linux/macOS)

Linux/macOS

基本用法

./bot.sh <命令> <参数>

支持的命令

命令 参数 描述
send <消息内容> 发送 Markdown 格式的文本消息
sendfile <文件路径> 发送文件到群聊

使用示例

发送文本消息

# 发送简单文本消息
./bot.sh send "服务器运行正常"

# 发送 Markdown 格式消息
./bot.sh send "### 系统警告\n请立即检查系统状态"

# 发送多行消息
./bot.sh send "### 新警告\n请立即检查系统状态。\n\n> 这是一条重要通知"

发送文件

# 发送当前目录下的文件
./bot.sh sendfile ./API.md

# 发送绝对路径文件
./bot.sh sendfile /home/user/documents/report.pdf

# 发送日志文件
./bot.sh sendfile /var/log/system.log

配置信息

默认机器人 ID： ${DEFAULT_BOT_ID}

默认安全码： ${SECURITY_CODE}

服务器地址： ${SERVER_URL}

权限设置

# 给脚本执行权限
chmod +x bot.sh

# 运行脚本
./bot.sh send "测试消息"

2. bot.bat (Windows)

Windows

基本用法

bot.bat <命令> <参数>

支持的命令

命令 参数 描述
send "消息内容" 发送 Markdown 格式的文本消息（需要用引号包围）
sendfile <文件路径> 发送文件到群聊

使用示例

发送文本消息

:: 发送简单文本消息
bot.bat send "服务器运行正常"

:: 发送 Markdown 格式消息
bot.bat send "### 系统警告\n请立即检查系统状态"

:: 发送多行消息
bot.bat send "### 新警告\n请立即检查系统状态。\n\n> 这是一条重要通知"

发送文件

:: 发送当前目录下的文件
bot.bat sendfile API.md

:: 发送绝对路径文件
bot.bat sendfile C:\Users\Username\Documents\report.pdf

:: 发送日志文件
bot.bat sendfile C:\logs\system.log

配置信息

默认机器人 ID： %DEFAULT_BOT_ID%

默认安全码： %SECURITY_CODE%

服务器地址： %SERVER_URL%

运行方式

:: 在命令提示符中运行
bot.bat send "测试消息"

:: 在 PowerShell 中运行
.\bot.bat send "测试消息"

Windows 二进制程序

bot.exe 是功能更强大的 Windows 二进制程序，提供完整的命令行工具功能。

高级用法

在自动化脚本中使用

Linux/macOS Cron 任务

# 每小时发送系统状态
0 * * * * /path/to/bot.sh send "$(date): 系统运行正常"

# 每日发送日志文件
0 0 * * * /path/to/bot.sh sendfile /var/log/daily.log

Windows 计划任务

:: 在批处理文件中使用
@echo off
echo 开始系统检查...
:: 执行系统检查命令
systeminfo > system_status.txt
bot.bat sendfile system_status.txt
echo 系统状态已发送到企业微信群

错误处理

注意：脚本会返回不同的退出码来表示执行结果：

  • 0：成功
  • 1：参数错误
  • 2：网络错误
  • 3：文件不存在

Linux/macOS 错误处理示例

#!/bin/bash
./bot.sh send "开始备份任务"
if [ $? -eq 0 ]; then
    echo "消息发送成功"
else
    echo "消息发送失败"
fi

Windows 错误处理示例

@echo off
bot.bat send "开始备份任务"
if %errorlevel% equ 0 (
    echo 消息发送成功
) else (
    echo 消息发送失败
)

常见问题解答

Q: 脚本无法执行怎么办？

Linux/macOS：确保脚本有执行权限：chmod +x bot.sh

Windows：确保在命令提示符或 PowerShell 中运行，或者双击运行

Q: 如何修改脚本中的配置？

A: 用文本编辑器打开脚本文件，修改顶部的配置变量：

# Linux/macOS (bot.sh)
DEFAULT_BOT_ID="你的机器人ID"
SECURITY_CODE="你的安全码"
SERVER_URL="http://your-server:8080"

# Windows (bot.bat)
set DEFAULT_BOT_ID=你的机器人ID
set SECURITY_CODE=你的安全码
set SERVER_URL=http://your-server:8080

Q: 如何发送包含特殊字符的消息？

Linux/macOS：使用单引号或转义字符：./bot.sh send 'message with "quotes"'

Windows：使用双引号并转义内部引号：bot.bat send "message with \"quotes\""

Q: 支持发送哪些类型的文件？

A: 支持企业微信机器人允许的所有文件类型，包括：

  • 文档类：PDF、DOC、DOCX、XLS、XLSX、PPT、PPTX
  • 图片类：PNG、JPG、JPEG、GIF
  • 压缩包：ZIP、RAR、7Z
  • 文本类：TXT、LOG、MD
  • 其他：符合企业微信机器人文件大小限制的文件

最佳实践

  • 定期检查脚本的执行权限和配置
  • 在生产环境中使用时，建议添加错误处理和日志记录
  • 合理控制消息发送频率，避免刷屏
  • 对于重要的自动化任务，建议同时记录本地日志
  • 定期测试脚本功能，确保服务器连接正常
